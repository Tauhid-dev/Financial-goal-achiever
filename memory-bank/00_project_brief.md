# Project Brief — Family Finance Intelligence Platform

## Vision
Build a privacy-first, modular financial intelligence platform for families.
Users can upload financial documents (starting with bank statements) and receive:
- Structured transactions
- Budget insights
- Savings goals & projections
- “What-if” simulations

## Target Users
- Families & parents
- Household budget managers
- Future: accountants, SMEs, open banking users

## Core Values
- Privacy-first (no raw document storage)
- Deterministic logic (rules before AI)
- Modular & extensible
- Safe for families and children
- Deployment-ready from day one

## v1 Scope
- PDF ingestion (bank statements)
- Transaction normalization
- Budget analysis
- Savings goals & projections
- API contracts only
- Containerized backend

## Explicit Non-Goals (v1)
- No UI
- No real bank integrations
- No LLM usage
- No background jobs
- No production deployment
